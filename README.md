# tag-pp

tag-pp:
    $ tag-pp <サブコマンド>

    サブコマンド一覧
    - tag-pp
    $ tag-pp tag-pp <tag.csvの場所> <出力先>
    tagをtag_pp.csv形式に変換

    - geotag-pp
    $ tag-pp geotag-pp <tag_pp.csvの場所> <geotag.csvの場所> <出力先>
    geotagをgeotag_pp.csv形式に変換

    - gen-test <tag.csv> <geotag.csv> <行数>
    テスト用の小さなtag_pp.csvとgeotag_pp.csvを作る
